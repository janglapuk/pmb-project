Tentang Proyek

Pendahuluan

Manajemen Penerimaan Mahasiswa Baru (PMB) adalah aplikasi berbasis website untuk memanajemen pendaftaran calon mahasiswa baru yang akan berencana kuliah di STMIK SPB Samarinda.

Latar Belakang

Tujuan pembuatan project ini sebagai pra-syarat mengikuti Ujian Akhir Semester STMIK SPB Samarinda. Selain itu juga sebagai sarana belajar dan sebuah tantangan baru dalam dunia pemrograman website berbasis pemrograman Python sebagai bahasa pemrograman yang direkomendasikan oleh STMIK SPB Samarinda.

Tim Pengembang

Adapun tim yang mengembangkan aplikasi ini adalah dari mahasiswa STMIK SPB Samarinda, terdiri dari:

  • Thami Rusdi Agus (2014.201.00104)
  • Edy Yansyah (2014.201.00105)
  • Rizky Rachmatini (2014.201.00106)

Informasi Teknis

Informasi teknis yang digunakan, yaitu:

  • Python
  • Django Web Framework
  • jQuery & AJAX
  • JSON
  • Bootstrap (oleh Twitter), sebagai front-end framework
  • OpenShift (oleh RedHat), sebagai penyedia layanan gratis penyedia hosting / PaaS (Platform as a Service)
  • GitHub, sebagai penyedia layanan SCM (Source Code Management)
